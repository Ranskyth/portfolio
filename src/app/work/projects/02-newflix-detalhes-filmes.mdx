## Visão Geral

Aplicação web desenvolvida para permitir que usuários explorem informações detalhadas sobre filmes. Integrada com API de filmes para exibir catálogo atualizado com avaliações, sinopses e dados relevantes.

## Tecnologias Utilizadas

- **Frontend**: Next.js para performance e SEO
- **Styling**: Tailwind CSS para design responsivo e utilitário
- **API**: Integração com API de filmes para dados em tempo real
- **Deployment**: Vercel para hosting e CI/CD automático

## Principais Funcionalidades

- Catálogo de filmes com busca e filtros
- Página de detalhes completa para cada filme
- Informações de avaliações e sinopse
- Design responsivo para todos os dispositivos
- Carregamento otimizado com Next.js Image

## Resultado

Projeto demonstrando proficiência em desenvolvimento frontend moderno, otimização de performance e integração com APIs externas.
