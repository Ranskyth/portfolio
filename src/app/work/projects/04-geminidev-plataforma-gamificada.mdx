## Visão Geral

Plataforma inovadora desenvolvida para transformar o aprendizado em uma experiência gamificada e engajante. Estudantes podem participar de desafios de programação, ganhar pontos e competir em rankings.

## Tecnologias Utilizadas

- **Frontend**: Next.js para interface moderna e responsiva
- **Backend**: Spring Boot com Hibernate para API robusta
- **Database**: MySQL para armazenamento escalável
- **DevOps**: Docker para containerização e deployment
- **Architecture**: Microserviços e arquitetura desacoplada

## Principais Funcionalidades

- Sistema de desafios e quests programáticas
- Gamificação com pontos, badges e rankings
- Painel de progresso e estatísticas do aluno
- Sistema de autenticação seguro
- API REST bem documentada
- Interface responsiva e intuitiva
- Integração frontend-backend otimizada

## Desafios Técnicos Resolvidos

- Arquitetura escalável para suportar múltiplos usuários
- Sistema de pontuação e ranking em tempo real
- Integração entre stack JavaScript (Next.js) e Java (Spring Boot)
- Gerenciamento eficiente de banco de dados relacional
- Containerização com Docker para fácil deployment

## Resultado

Projeto fullstack completo demonstrando capacidade de trabalhar com múltiplas tecnologias, desde frontend moderno até backend em Java com Spring Boot. Showcasa habilidades em architecture, integração de sistemas e desenvolvimento de features complexas.
