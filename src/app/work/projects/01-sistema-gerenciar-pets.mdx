## Visão Geral

Sistema desenvolvido para facilitar o gerenciamento de pets, permitindo que donos registrem e acompanhem todas as informações dos seus animais de estimação. A plataforma integra frontend moderno com backend robusto e banco de dados escalável.

## Tecnologias Utilizadas

- **Frontend**: Next.js com componentes reutilizáveis
- **Backend**: Nest.js para API escalável e bem estruturada
- **Database**: Prisma ORM para gerenciamento de dados
- **DevOps**: Docker para containerização e deployment

## Principais Funcionalidades

- Registro e perfil completo de pets
- Histórico de vacinas e consultas veterinárias
- Acompanhamento de medicamentos
- Lembretes de cuidados
- Integração entre frontend e backend
- Interface responsiva e intuitiva

## Resultado

Aplicação funcional pronta para produção, demonstrando capacidade de trabalhar com stack fullstack moderno, desde a camada de apresentação até infraestrutura containerizada.
