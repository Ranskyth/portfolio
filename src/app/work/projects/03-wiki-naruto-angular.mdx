## Visão Geral

Aplicação desenvolvida com Angular que funciona como uma wiki interativa dos personagens da série Naruto. Permite explorar informações sobre personagens, suas habilidades e histórias.

## Tecnologias Utilizadas

- **Frontend**: Angular para SPA dinâmica e reativa
- **Styling**: CSS moderno com design responsivo
- **Deployment**: Vercel para hosting
- **Data**: Estrutura modular para gerenciamento de dados de personagens

## Principais Funcionalidades

- Catálogo completo de personagens
- Busca e filtros por nome e atributos
- Página individual para cada personagem com detalhes
- Interface responsiva e navegação intuitiva
- Componentes reutilizáveis em Angular

## Aprendizados

Projeto que demonstra experiência com framework Angular, state management e construção de SPA (Single Page Applications) completas.

## Resultado

Aplicação funcional que showcasa habilidades em Angular e design de interfaces interativas para dados estruturados.
